

# AI-Powered Obstetric Emergency Decision Support System

## Overview

This project is an AI-powered clinical decision support prototype designed to assist healthcare professionals in assessing obstetric emergency risk levels.

The system evaluates patient symptoms and clinical indicators to estimate the severity of an obstetric emergency and provide evidence-based recommendations.

This project is inspired by our published systematic review:

**"Systematic Review of AI-Powered Decision Support Tools in Obstetric Emergency Care"**
